# luaxx - An easy to use Lua <-> C++ binder

"luaxx" can be used to communicate with Lua in a C++ project in an intuitive way.

### Dependency

Requires a C++ compiler with variadic template (since C++11) capability.

### Usage

luaxx is widely used in [Bitty Engine](https://github.com/paladin-t/bitty), see [the scripting API binding](https://github.com/paladin-t/bitty/blob/main/src/scripting_lua_api.cpp) for usages.
